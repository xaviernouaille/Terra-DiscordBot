# Terra-DiscordBot

## Fonctionnalité

- Envoi des offres d'emploi de la journée sur un channel Discord

## Commande

```node
npm run start
npm run dev
```

## Modules utilisés

```json
    "axios": "^0.20.0",
    "cheerio": "^1.0.0-rc.3",
    "discord.js": "^12.3.1",
    "dotenv": "^8.2.0",
    "nodemon": "^2.0.4"
```
